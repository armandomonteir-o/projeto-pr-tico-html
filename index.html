O que é o HTML?

Vale frisar que, para começar, temos que deixar bem claro que o HTML não é uma linguagem de programação, e sim, uma linguagem de marcação.

o HTML é usado na criação de paginas Web e seus documentos são interpretados pelos Navegadores.

Resumidamente, é responsável por toda a estrutura do site que nós (usuários) conseguimos enxergar.

Tags mais utilizadas

O HTML é uma linguagem de marcação que permite criar páginas web com conteúdo e estrutura definidos. Para isso, o HTML usa tags, que são elementos que indicam ao navegador como interpretar e exibir o conteúdo da página. Existem muitas tags HTML, mas algumas são mais utilizadas do que outras. Neste texto, vamos ver quais são as tags mais usadas no HTML e para que servem.

Tags estruturais

As tags estruturais são aquelas que definem a estrutura básica de um documento HTML. Elas são:

  • "!DOCTYPE html" Esta tag informa ao navegador que o documento é do tipo HTML5, a versão mais recente do HTML
  • "HTML": Esta tag define o início e o fim do documento HTML. Ela pode ter o atributo lang, que indica o idioma principal do documento
  • "Head": Esta tag contém informações sobre o documento, como o título, a codificação de caracteres, os metadados, os links e os scripts
  • "Link": Esta tag define uma referência a um recurso externo, como uma folha de estilo, um ícone, uma fonte, etc. Ela pode ter vários atributos, como rel, href, type, etc
  • "Body": Esta tag define o corpo da página, onde fica o conteúdo principal que será exibido ao usuário

Tags de conteúdo

As tags de conteúdo são aquelas que definem o conteúdo que será exibido na página, como textos, imagens, links, listas, tabelas, formulários, etc. Elas são:

  • "h1 a h6: Estas tags definem cabeçalhos de diferentes tamanhos, sendo h1 o maior e h6 o menor. Elas são usadas para organizar o conteúdo em seções e subseções, e também para melhorar a acessibilidade e a semântica da página
  • "p": Esta tag define um parágrafo, que é um bloco de texto separado por uma quebra de linha. Ela é usada para formatar o texto em unidades lógicas
  • "ol e ul": Definem uma lista, que é um elemento que exibe uma série de itens. A tag "ul" define uma lista não ordenada, onde os itens são marcados com pontos ou outros símbolos. A tag "ol define uma lista ordenada, onde os itens são numerados ou alfabeticamente. Ambas as tagas devem conter a tag "li" que define cada item da lista

    • Como o HTML impacta no nosso dia a dia?

    O HTML impacta no nosso dia a dia de várias formas, pois é a base para a criação de sites que usamos para nos informar, nos comunicar, nos divertir, nos educar, entre outras coisas. Alguns exemplos de como o HTML influencia o nosso cotidiano são:

    1. O HTML permite que possamos ler notícias, artigos, blogs e outros conteúdos online, usando elementos como parágrafos, títulos, listas, imagens, links, etc
    2. O HTML possibilita que possamos interagir com sites e aplicativos web, usando elementos como formulários, botões, caixas de texto, menus, etc
    3. O HTML facilita que possamos acessar sites adaptados para diferentes dispositivos, como celulares, tablets e computadores, usando elementos como meta tags, media queries, viewport, etc
    4. O HTML integra-se com outras tecnologias, como CSS e JavaScript, que permitem adicionar estilo e funcionalidade aos sites, criando experiências mais ricas e dinâmicas para os usuários
